Time to lock it down! By default, Streamlit's port (8501) is blocked. Let's open it *only* for your home IP address. ğŸ›¡ï¸

### 1. Create the Firewall Rule ğŸ§±

In the Cloud Shell, run this command. **Remember to replace `[YOUR_HOME_IP]` with your actual IP address and add `/32` to the end!**

```bash
gcloud compute firewall-rules create allow-streamlit-home-only \
    --direction=INGRESS \
    --priority=1000 \
    --network=default \
    --action=ALLOW \
    --rules=tcp:8501 \
    --source-ranges=[YOUR_HOME_IP]/32 \
    --target-tags=streamlit-server

```

